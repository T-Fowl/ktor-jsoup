HttpClient plugin that parses response bodies into Jsoup Document class using a provided Parser

By default,

ContentType.Text.Html is parsed using Parser.htmlParser.

ContentType.Text.Xml&ContentType.Application.Xml are parsed using Parser.xmlParser.

Note: It will only parse registered content types and for receiving Document or superclasses.
